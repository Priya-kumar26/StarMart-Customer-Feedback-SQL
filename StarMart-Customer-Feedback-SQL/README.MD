## How to run locally

1. Download the `.sql` files provided in this repository.
2. Open an online (e.g., SQL Fiddle) or local SQL Server environment.
3. Paste and execute the files in this order:
   - `create_tables.sql`
   - `insert_sample_data.sql`
   - `important_queries.sql`

## Files Included

- `create_tables.sql`: SQL commands to create all required tables
- `insert_sample_data.sql`: Sample data for customers, products, stores, purchases, and feedback
- `important_queries.sql`: Analytics and reporting queries demonstrating SQL skills
